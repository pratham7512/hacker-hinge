## Setup
- Copy `.env.example` to `.env.local` and fill values: `DATABASE_URL`, `GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET`, `NEXTAUTH_URL`, `NEXTAUTH_SECRET`.
- Install deps: `pnpm i`
- Generate Prisma client: `pnpm prisma generate`
- Dev: `pnpm dev`
